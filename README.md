# 儿童书籍动画化系统

## 项目简介

儿童书籍动画化系统是一个将童话、故事或科普书籍转换为动画和图片的网页应用，旨在帮助小朋友更生动地阅读书籍内容。系统支持用户上传书籍、生成相关插图、添加动画效果，并提供用户认证和权限管理功能。

## 功能特性

- **用户认证**：支持用户登录、注册和权限管理
- **书籍管理**：添加、编辑、删除书籍
- **书籍导入**：支持PDF和Word文档导入
- **自定义页面**：支持为书籍创建自定义页面
- **AI图像生成**：根据书籍内容生成相关插图
- **动画效果**：为生成的图片添加动画效果
- **图片管理**：保存生成过的图片，支持选择多个图片
- **管理员功能**：邀请新用户、设置用户权限

## 技术栈

- **前端**：HTML5, CSS3, JavaScript
- **后端**：Node.js, Express
- **数据存储**：JSON文件存储
- **认证**：JWT（JSON Web Token）
- **文件处理**：multer（文件上传）、pdf-parse（PDF解析）、mammoth（Word文档解析）
- **AI图像生成**：使用文本转图像API
- **动画效果**：Web Animations API

## 安装和使用

### 安装步骤

1. 克隆项目到本地
2. 安装依赖：`npm install`
3. 启动服务器：`npm start`
4. 访问：http://localhost:3000

### 默认账号

- **管理员账号**：admin@example.com / admin123

### 使用方法

1. **登录系统**：使用默认管理员账号或注册新用户
2. **添加书籍**：
   - 手动输入书名和内容
   - 或上传PDF/Word文档导入
   - 或使用自定义页面创建
3. **管理书籍**：在书籍列表中查看、编辑或删除书籍
4. **查看书籍**：
   - 点击书籍列表中的"查看"按钮
   - 使用"上一页"和"下一页"按钮翻页
   - 点击"生成图片"按钮为当前页面生成插图
   - 点击"添加动画"按钮为生成的图片添加动画效果
   - 在图片选择区域选择之前生成的图片
5. **用户管理**（管理员）：
   - 邀请新用户
   - 修改用户权限
   - 删除用户

## 注意事项

- 系统使用JSON文件存储数据，适合小型应用
- AI图像生成功能需要网络连接
- 上传的文件会被临时存储，解析完成后会被删除
- 请确保服务器有足够的存储空间和处理能力
